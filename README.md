# Sistema de Robôs de Trading Automatizado – MQL5 & Python

Este repositório tem como objetivo centralizar o desenvolvimento colaborativo de **robôs de trading automatizados**, utilizando as linguagens **MQL5** (MetaQuotes Language 5) e **Python**.

Nosso foco está na construção de **estratégias quantitativas**, **modelos de execução automatizada** e **ferramentas auxiliares** voltadas para operação nos mercados financeiros por meio da plataforma **MetaTrader 5 (MT5)**.

---

## 📌 Objetivos

- Desenvolver e otimizar Expert Advisors (EAs) em **MQL5**
- Realizar análises quantitativas com **Python**
- Integrar estratégias via **MetaTrader5 API para Python**
- Validar estratégias com testes históricos (backtests)
- Promover aprendizado técnico entre os participantes

---

## 🧰 Tecnologias Utilizadas

| Tecnologia      | Finalidade                                      |
|------------------|--------------------------------------------------|
| **MQL5**          | Programação de robôs (EAs) na plataforma MT5     |
| **Python**        | Análise de dados, integração, sinalização        |
| **Pandas/NumPy**  | Manipulação de dados financeiros                 |
| **Matplotlib**    | Visualização de resultados e backtests           |
| **MetaTrader5 API** | Execução e controle externo de operações       |
